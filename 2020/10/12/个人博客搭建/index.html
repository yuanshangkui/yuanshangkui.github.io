个人博客搭建

搭建本地环境

  1. 下载node.js到电脑，安装

  2.  node -v        查看nodejs版本
 
  3.  npm install -g cnpm --registry=https://registry.taobao.org 
 全局（-g）安装指向淘宝镜像源，加速
 
  4.  hexo -v                查看是否安装完成
 
  5. hexo server        启动博客
    本地 http://localhost:4000   预览博客文章

  6. markdown写博客

  7. hexo clean        清理博文
hexo generate  （或者 hexo g）    生成博文

  8. 设置F:\blog目录下的 _config.yml文件

  9. 部署到远端
